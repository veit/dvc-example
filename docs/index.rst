===========
DVC-Example
===========

`DVC <https://dvc.org//>`_ erleichtert die systematische Organisation der
Modellentwicklung für maschinelles Lernen (ML). Dieses Beispiel versucht zu
verdeutlichen, wie Ihr Euer Modell mit Parametrisierung verbessern könnt, sobald
mehr Trainingsdaten zur Verfügung stehen. Das Beispiel zeigt auch, wie Ihr die
Verbesserungen nachvollziehbar messen könnt.
